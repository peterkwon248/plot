## after-work: 작업 종료 정리

아래 순서대로 실행해줘:

1. 이번 세션에서 대화 내용을 분석해서 **진행된 작업**을 정리해줘
2. git log와 git diff로 실제 코드 변경사항을 확인해줘
3. 아래 형식으로 종료 리포트를 출력해줘:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 AFTER WORK 종료 리포트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[오늘 완료한 작업]
- ___
- ___

[진행 중 (미완료)]
- ___

[다음에 해야 할 일] (우선순위 순)
1. ___
2. ___
3. ___

[핵심 결정사항 / 메모]
- ___

[프로젝트 상태]
- 커밋 수: ___
- 변경된 파일: ___

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

4. `.claude/session-log.md` 파일을 위 내용으로 **업데이트**해줘
5. "수고하셨습니다. 오늘은 여기까지." 로 마무리
