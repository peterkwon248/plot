## before-work: 작업 시작 브리핑

아래 순서대로 실행해줘:

1. `.claude/session-log.md` 파일을 읽어서 **이전 세션 요약**을 보여줘
2. 프로젝트 현재 상태를 파악해줘:
   - git status로 변경사항 확인
   - 주요 파일 구조 훑기
3. 아래 형식으로 브리핑을 출력해줘:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 BEFORE WORK 브리핑
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[지난 세션 요약]
- 마지막 작업일: ___
- 완료한 것: ___
- 핵심 결정사항: ___

[오늘 해야 할 일]
- 우선순위 1: ___
- 우선순위 2: ___
- 우선순위 3: ___

[현재 프로젝트 상태]
- 브랜치: ___
- 미커밋 변경: ___
- 주의사항: ___

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

4. "어디서부터 시작할까요?" 로 마무리
